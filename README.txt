#####     Spider-Box     ######
Game Rules:
    The objective of this game is to kill Mr.Red and get a powerup.
After killing Mr.Red the victory door spawns.
Do this without touching the ground and you'll win.
Screen turns red for defeat and blue for victory.

Controls:
    To restart press the RETURN key.
    To quit game press the ESC key.

How to run:
    To run the game open up a terminal window in the same
directory as the project2.py and run that file
